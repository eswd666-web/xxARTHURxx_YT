# Bloxd.io T-Packs Download Site

A free website for downloading Bloxd.io Texture Packs (T-Packs).

## Features

- 🎨 Beautiful and responsive design
- 📦 Easy T-pack browsing and downloading
- 📱 Mobile-friendly interface
- 🚀 Fast and lightweight

## How to Add T-Packs

1. Open `script.js`
2. Add your T-pack to the `tpacks` array with:
   - `name`: T-pack name
   - `description`: Brief description
   - `icon`: Emoji icon
   - `downloadUrl`: Direct download link to your T-pack file

## Example

```javascript
{
    id: 5,
    name: "Your Pack Name",
    description: "Description of your pack",
    icon: "🎮",
    downloadUrl: "https://example.com/your-tpack.zip"
}
```
